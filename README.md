# Vector OS

Open-source organization focused on cybersecurity, Linux, developer tools, and modern infrastructure.

## About

Vector OS is a community-driven organization building open-source projects for developers, hackers, engineers, and builders.

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- Node.js

## Getting Started

```bash
git clone https://github.com/Vector-OS/vector-os-web.git

cd vector-os-web

npm install

npm run dev
```
